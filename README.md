# SCDDP
SCDDP 意指 用 Scratch 環境所寫的 對對碰(DDP)對戰題庫遊戲。

原本想在此提供 [可直接測試] 的連結，但此處的機制似乎不讓程式內可以正常載入外部檔案，所以就不提供直接測試了。

但是可以將整個題庫系統的架構，用此處提供的功能： [Download ZIP] 下載，下載回去之後再另外找能放 HTML 檔的網路空間或模擬網路空間的程式，就可以自行測試。

下載之後，解壓縮會有一個 scddp-main 的資料夾，裡面有：

------------------------------------------------------------------------------------------

.github：這個資料夾不需要留著

Chem_Elem_Pic：化學元素符號_題庫資料夾(小圖回答小圖的題型範例)

Chem_Tools：化學器材名稱題_庫資料夾(小圖回答大圖的題型範例)

End_V1200_A01_V2EC：高雄澎湖灣敏雄老師提供的英文測試用題庫資料夾(文字與語音的混合題型)

JVoc_A1：日文單字題庫資料夾(小圖與語音的混合題型)

_list.txt：題庫選單的參數檔

BK_1.mp3：遊戲背景音樂

Boing.mp3：遊戲音效

BOMBOM.mp3：遊戲音效

SCDDP_0121a_Web.html：[主程式]

------------------------------------------------------------------------------------------

所謂 [模擬網路空間] 的程式：
